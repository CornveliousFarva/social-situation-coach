# Social Situation Coach

A React and TypeScript web application designed to help users prepare for unfamiliar social environments.

## Features

- Religious or spiritual gathering guidance
- Sporting event preparation
- Professional networking event coaching
- Conversation starters
- Social norms and etiquette guidance
- Exit strategies
- Anxiety-based recommendations

## Technologies

- React
- TypeScript
- Vite
- Tailwind CSS

## Installation

```bash
npm install
npm run dev
